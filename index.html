Header-generator

Create or modify header file with all function found in all .c files in current directory (pwd) and all subdirectories

View on GitHub Download .zip Download .tar.gz

You are too lazy to search in your .h file(s) and modify it for one function ?

Great ! I have a solution for you !

Presentation

Header generator is a tool that adds at the end of .h passed in argument all prototypes of functions found (except static ones) in .c files in your pwd and in all it's sub-directories. Linker is no longer a slut.
